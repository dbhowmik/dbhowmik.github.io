Competition: Design and Develop your Hardware/Software application

Design and Develop Your Own Hardware / Software

What do you need to do

  • Design and Development of an application of your choice on either small hardware / smart phone......
  • Hardware: Raspberry Pi, Pi Zero, Arduino Uno, Intel Galileo, BeagleBone / Any android smart phone etc...
  • Software: OS - Linux
  • Development language: C/C++, Java, Python .. any

Examples

  • Media centre
  • Create a wireless extender
  • Person / Object identifier
  • Recognise faces
  • E-Com website
  • Any thing interesting you can think of

Rules

  • Open to 1st year undergraduate students
  • Group of up to 3 individual members (single person group is possible)
  • Need to demonstrate, write a short report / doc, provide code
  • Registration to deepayan.bhowmik@shu.ac.uk by 15th March 2017.
  • Deadline to submit code and report: 28th April 2017
  • Prize: Winner: £150 cash / voucher, Runner: £100 cash / voucher
  • Funding acknowledgement: Learning and Teaching Enhancement Grant, Sheffield Hallam Univeristy.
